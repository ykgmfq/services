# Service definitions

Self-hosted services running as Podman Quadlets on a single Fedora host.
The domain is `dm-poepperl.de`.
This repository holds the image build contexts and the systemd Quadlet definitions for the whole stack, so the host can be rebuilt from source.

For agent and contributor context see [CLAUDE.md](CLAUDE.md).
For host system setup see https://github.com/ykgmfq/ansible.
For layout, configuration, and usage see the GitHub wiki.
